Why it exists

A collector tool that keeps the collection yours.

Pokevault combines a scraped Pokedex reference, a local REST API and a fast vanilla web app. It is built for people who want to organize real collections without giving their data to a hosted tracker.

Pokedex-first

Cherche, J'ai and Double stay centered on each Pokemon entry, with Vu chez and Match layered in from optional Dresseurs.

Cards as enrichment

Attach owned cards to Pokemon without turning the app into a market tracker.

Binder-aware

Model physical binders, grids, pages and printable checklists.

Local exchanges

Export Trainer Cards manually, import received cards into a searchable contact book and keep every file local.

Screenshots

One workspace for browsing, organizing and measuring progress.

Quick start

Run it locally in a few commands.

git clone https://github.com/Boblebol/pokevault.git
cd pokevault
make install
make dev
