Product surface

Features

Pokevault is built around the Pokedex, then layers binders, local Trainer Cards, badges and local-first data workflows on top. App documentation also lives inside the local app at #/docs.

Collection workspace

Browse the National Pokedex with region chips, narrative tags, fuzzy search, keyboard shortcuts and compact Caught, Double and Release actions backed by local progress JSON.

Pokemon modal

Open one modal for identity, artwork, Caught, Double and Release actions, linked forms, personal notes, type matchups and game Pokedex appearances from data/game-pokedexes.json.

Game Pokedex appearances

See which regional game Pokedex definitions reference a Pokemon, starting from a compact local data file that can grow without adding new user state.

Physical binder planner

Create local binders with a 3×3 · 10 sheets default, compact family rows such as Spoink / Grumpig / Spinda with strict alignment, regional splits such as Kanto 1 and Kanto 2. Large 3×3 ring binder mode keeps one physical binder with 3×3 recto-verso sheets, region sections that start on a new sheet front (recto), compact families per region, regional forms in their form region and auto capacity with 10 spare sheets.

Trainer Cards

Create a local card with optional Instagram, Facebook or phone contact links, export it as a file, import received Dresseurs and keep a searchable local contact book with Double and Vu chez context, without accounts or server sync.

Badges

Unlock Souvenirs de Kanto badges for Rouge/Bleu champions d'arene, Conseil 4 and rival teams, Or/Argent for Johto, Kanto, Peter and Silver, then base-version teams sans remakes from Rubis/Saphir, Diamant/Perle, Noir/Blanc, Noir 2/Blanc 2, X/Y, Soleil/Lune, Epee/Bouclier and Ecarlate/Violet. The badge gallery keeps sealed badges until unlock and stays separate from collection statistics.

Stats

Track global, regional and type Pokedex progress on a page dedicated to collection numbers.

Local-first data

Keep one local collection state in readable JSON files, then export or import the full collection as a versioned backup.

Product tour

Start with a compact five-step tour that explains Caught, Double, Dresseurs, manual imports and local files before asking for region and tracking density.

App documentation

Open #/docs in the local app for a complete bilingual product guide to setup, workflows, local files, Trainer Cards, badges, shortcuts and API endpoints.

Local REST API

Use the local FastAPI surface for progress, badges, trainers, binders, export and import integrations, plus static game Pokedex reference data.

Pokedex-first stance

Pokemon entries stay centered on capture.

Version 1.7.0 keeps completion workflows first: dashboard, capture, duplicates, filters and Pokemon fiches all prioritize Pokedex state. Marketplace pricing, public wishlists and hosted accounts remain outside the active scope.
