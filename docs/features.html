Product surface

Features

Pokevault is built around the Pokedex, then layers physical cards, binders, local Trainer Cards, badges and local-first data workflows on top. App documentation also lives inside the local app at #/docs.

Collection workspace

Browse the National Pokedex with region chips, narrative tags, fuzzy search, keyboard shortcuts and compact Caught, Double, Release 1 and Release actions backed by local progress JSON.

Pokemon fiches

Open a drawer or full page for identity, artwork, Caught/Double/Release 1/Release actions, linked forms, personal notes, type matchups and owned cards.

Physical card catalog

Attach set, number, variant, language, condition, quantity, notes and image URLs to a Pokemon, with public Pokemon TCG API search for metadata prefill.

Physical binder planner

Create local binders with a 3×3 · 10 feuillets default, custom grids, regional splits such as Kanto 1 and Kanto 2, and optional Familles layouts that keep evolution stages aligned with intentional empty slots.

Trainer Cards

Create a local card with optional Instagram, Facebook or phone contact links, export it as a file, import received Dresseurs and keep a searchable local contact book with Double and Vu chez context, without accounts or server sync.

Badges

Unlock badges as you capture Pokemon and keep the gallery in Stats, without active missions or forced focus.

Badges and stats

Track global, regional, type, card and badge progress, including Souvenirs de Kanto badges for Rouge/Bleu champions d'arene, Conseil 4 and rival teams, plus Or/Argent badges for Johto, Kanto post-game, Peter and Silver. The badge gallery keeps sealed badges until unlock with Pokemon requirement previews. The base-version nostalgia catalog continues sans remakes with Rubis/Saphir, Diamant/Perle, Noir/Blanc, Noir 2/Blanc 2, X/Y, Soleil/Lune, Epee/Bouclier and Ecarlate/Violet.

Profiles and local-first data

Maintain separate local Pokedex profiles for main, casual, shiny-only or other workflows, then export or import the full local collection state as versioned JSON.

Print view

Generate printable binder, regional or pocket A5 checklists from the same collection and binder data used by the app.

Product tour

Start with a compact five-step tour that explains Caught, Double, Release 1, Release, Dresseurs, manual imports and local files before asking for region and tracking density.

App documentation

Open #/docs in the local app for a complete bilingual product guide to setup, workflows, local files, Trainer Cards, badges, shortcuts and API endpoints.

Local REST API

Use the local FastAPI surface for progress, cards, badges, profiles, trainers, binders, export and import integrations.

Pokedex-first stance

Cards enrich Pokemon entries; they do not replace them.

Version 1.4.0 keeps completion workflows first: dashboard, capture, duplicates, filters and Pokemon fiches all prioritize Pokedex state before card inventory. Marketplace pricing, public wishlists and hosted accounts remain outside the active scope.
