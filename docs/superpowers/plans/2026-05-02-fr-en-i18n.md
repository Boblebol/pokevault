# FR/EN Switch Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a lightweight French/English switch for the landing and app, with French as the default and no dependency.

**Architecture:** Add small i18n runtimes for docs and app. Keep French fallback text in HTML, translate through `data-i18n*` attributes and expose `t()` for dynamic modules.

**Tech Stack:** Plain browser JavaScript, static HTML, Node built-in test runner, Pytest docs checks.
