Product direction

Roadmap

The public RICE roadmap is complete through Wave 4. New ideas are separated from the active roadmap until they have a clear local-first product case.

The v1.1 Pokédex-first home and richer Pokémon fiches are now shipped. The next active track is carefully sourced Pokédex metadata.

Wave 1 - Polish

Empty states, regional chips, multi-level progress and fuzzy keyboard search.

Wave 2 - Activation

Onboarding, enriched Pokedex status and narrative filters.

Wave 3 - Fiches

One Pokemon modal, local notes, type matchups and game Pokedex appearances.

Wave 4 - Delights

Artwork switcher, badges, profiles, pocket print and regional themes.

v1.5.0

Pokédex-first workflows now shipped.

Completion home

Dashboard, capture state and quick filters keep Pokédex progress visible first.

Richer fiches

A single modal shares status controls, linked forms, notes, type matchups and game Pokedex appearances.

Mobile fiche

Status, forms and notes stay reachable in the same modal on narrow screens.

Next local-first track

Dresseurs stay optional.

The local Trainer Card direction keeps the base Pokédex friction low: Capturé, Double, Relâcher 1 and Relâcher cover the local collection, while imported Dresseurs share only duplicates and can add Vu chez context without accounts, public profiles or automatic sync. The product tour introduces this model as a manual file exchange, so the feature stays discoverable without becoming mandatory.

v0.1 local card

Create, edit and export your own Trainer Card with optional contact links from the Dresseurs tab, without an account.

v0.2 received contacts

Import cards from other collectors, update by stable trainer identity, search the local book, keep private notes and delete stale contacts.

v0.3 trade context

Mark Double on Pokemon, import other trainers' duplicate lists, then show Vu chez hints for missing Pokemon without changing collection state automatically.

Not active

Known but postponed.

Social sharing, public profiles, market prices, daily streaks and scraper-heavy Pokedex descriptions are intentionally outside the active roadmap.
